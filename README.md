# Simple Login Page UI

## About Project

This is a simple Login Page UI created with HTML and CSS with Flexbox layout.
This UI has a left section with images and a right section with a login form.

---

## Technologies Used

* HTML5
* CSS3
* Flexbox

---

## What I Learned (By Fixing My Mistakes)

* Learned how to properly link CSS file with HTML
* Learned that CSS and HTML must be in separate files
* Fixed flexbox alignment issues in left and right sections
* Learned how flexbox layout arranges form elements vertically with flex-direction: column
* Fixed layout issues caused by wrong file structure
* Learned how to use full screen layout with height: 100vh
* Learned how to write clean and readable code

---

## How to Run

1. Open the project folder
2. Open **index.html** with browser

---

This project helped me understand basic frontend layout and how a beginner might go wrong with HTML and CSS.
